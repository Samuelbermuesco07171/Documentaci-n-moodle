codigo:002 nombre:VALIDACION DE USUARIOS

descripcion:Manual para validar si un usuario exite en el sistema

elementos necesarios:
-direccion url:https://fendipetroleo.com/Virtual/admin/user.php
-usuario administrador o gestor

datos necesarios para la creacion:
-nombre:opcional	
-apellido:opcional
-cedula:opcional, el numero de cedula al ser asignado tambien como usuario sirve en las dos busquedas
-email:opcinal	

proceso: 
1-se debe logear en el sistema con un usuario administrador o gestor, el cual debe tener la opcion de administracion de sitio, ver(cod.001 ima.001).
2-Despues de verificar que puede ingresar a la adminstracion de sitio debe ingresar por la ruta Área personal/Administración del sitio/Usuarios/Cuentas/Examinar lista de usuarios.
3-En esta pantalla encuentras el panel de busqueda de usuarios, como busqueda predeterminada lo podemos realizar por nombre de usuario(cedula).
